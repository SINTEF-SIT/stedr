A collection is a set of related cultural stories. For instance, stories in a collection may relate to a theme or to a neighbourhood. A collection consists of:

  1. an overall description telling what is the collection about
  2. a set of stories

The overall description is described in a similar way as any other cultural story using Digitalt fortalt, except that:

  1. The story should not relate to a place! You should not add any geo-location
  2. You should add the keyword (“Emneord”) “stedr_collection” to the story

The stories in the collection are described in a similar way as any other cultural story, with an additional rule:

  1. You should add the keyword “collection_. For instance, to add a story to the collection “Institutions in Kalvskinnet”, the keyword “collection_institutions_in_kalvskinnet” is added.

Note that any storyteller can thus add a new story to an existing collection. For more information about creating stories, look at “Creating stories for stedr”.
