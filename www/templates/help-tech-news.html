stedr uses several platforms on the Internet. This post describes shortcomings and changes in these platforms that affect stedr. If you notice that a feature ceases to work properly, give us feedback as a comment to this post.

  1. June 2014 - SoundCloud app: The latest version of SoundCloud app on iPhone (version 3.0) does no longer support recording. SoundCloud recommends you to use the AudioCopy application
  2. April 2014 - Retrieving places from Flickr: Places are currently stored in Flickr. The interface to Flickr is unstable and occasionally not available. This is why stedr asks you to refresh the map. We are looking for an alternative to Flickr
